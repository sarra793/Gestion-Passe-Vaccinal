# VACCIN - Gestion du Passe Vaccinal

## Description
Ce projet consiste à créer un programme permettant de gérer les informations vaccinales des citoyens et de déterminer leur éligibilité au téléchargement du passe vaccinal.

## Fonctionnalités
- Saisie du nombre de citoyens (5 à 100)
- Enregistrement des informations vaccinales
- Vérification de la validité des données
- Extraction du code, vaccin et nombre de doses
- Vérification du schéma vaccinal :
  - 2 doses reçues
  - ou 1 dose du vaccin Johnson
- Affichage du résultat pour chaque citoyen

## Structure du programme

### `saisir()`
Permet de saisir le nombre de citoyens.

### `remplir()`
Remplit le tableau des citoyens avec vérification des informations.

### `afficher()`
Analyse les données et affiche si le citoyen peut obtenir son passe vaccinal.
